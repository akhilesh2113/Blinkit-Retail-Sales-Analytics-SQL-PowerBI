# 🛒 Blinkit Sales Analysis | SQL + Power BI

An end-to-end Business Intelligence project built using **SQL** and **Power BI** to analyze Blinkit's retail sales data and convert it into meaningful business insights through an interactive dashboard.

---

# 📌 Project Overview

The aim of this project was to analyze Blinkit's retail sales data and understand how different products, outlets, and customer preferences contribute to overall business performance.

The project began with raw sales data, which was cleaned and transformed using SQL. After preparing the data, Power BI was used to build an interactive dashboard that allows users to explore the data through filters, charts, and KPIs.

The dashboard is designed to make business analysis simple and help users quickly answer common business questions without manually working with spreadsheets.


# 🎯 Business Objective

This dashboard helps answer questions such as:

* How much total revenue has the business generated?
* Which product categories generate the highest sales?
* Which outlet types perform better?
* Does outlet size impact sales performance?
* Which outlet locations contribute the most revenue?
* How do customer ratings vary across products?
